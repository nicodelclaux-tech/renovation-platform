# Product Spec

## Product thesis

A renovation image generator becomes genuinely useful only when it remembers the house. The system should preserve architectural constraints and iterate on creative variables.

## MVP scope

- Projects
- Rooms
- Style bible
- Room briefs
- Asset uploads
- Image generation
- Generation history
- Approved direction

## Non-MVP

- Revit integration
- 3D walkthroughs
- Supplier marketplace
- Budgeting
- Collaboration permissions
